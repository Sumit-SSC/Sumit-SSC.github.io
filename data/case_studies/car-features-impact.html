This project explains used-car price drivers using exploratory analysis and stakeholder-friendly dashboards. The emphasis is clarity: what matters, how much it matters, and how non-technical teams should use the findings.

Executive Summary

  • Top drivers: mileage, vehicle age, brand segment (premium vs mass), and fuel type dominate price variation.
  • Deliverable: Power BI + Excel dashboards that let users slice by brand, fuel, and mileage bins.
  • Outcome: stakeholders can price listings and negotiate with evidence instead of intuition.

Problem

Price discussions are often subjective. Teams need a repeatable view of how features correlate with price so they can set realistic expectations and understand trade-offs (e.g., older premium brands vs newer mass-market cars).

Analysis Approach

  1. Cleaning. handle missing values/outliers; standardize categorical values (brand, fuel).
  2. EDA. distributions + correlations + key bivariate relationships (price vs mileage/age).
  3. Segmentation. premium vs mass-market, mileage bins, and brand/fuel slices to reveal non-linear patterns.
  4. Dashboarding. package insights into interactive views for exploration and decision-making.

Insights (examples)

  • Mileage effect: strong negative relationship with price; slope differs by brand segment.
  • Age effect: depreciation is steep early and flattens later for some segments.
  • Fuel type: consistent price differences after controlling for other attributes.

How stakeholders use it

  • Pricing guidance: choose realistic ranges by segment and feature profile.
  • Listing strategy: highlight features that create measurable price lift.
  • Negotiation support: justify price changes with evidence-backed drivers.

Next Steps

  • Partial effects: add regression/ML to separate correlated drivers.
  • Geo/seasonality: incorporate region and time for market shifts.
  • Governance: metric definitions + data freshness checks for production dashboards.
